## 🎯 Objetivo
Este projeto foi criado para ajudar usuários a estimar:
- O consumo mensal de energia elétrica de um aparelho (em kWh).
- O custo aproximado da energia (com base em um valor fixo por kWh).

## 🛠️ Tecnologias
- **Linguagem:** Python 🐍
- **Plataforma:** GitHub 🌐

## 📐 Fórmula utilizada
O cálculo segue a fórmula:



\[
ConsumoMensal = \frac{Potência \times HorasDia \times 30}{1000}
\]



👉 Exemplo: uma geladeira de 50 W usada 30h/mês consome **45 kWh/mês**.

## 🚀 Como executar
1. Se você estiver no Windows: python app.py
2. ou, em alguns casos: py app.py
3. Se você estiver no Linux/Mac: python3 app.py
